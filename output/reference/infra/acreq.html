Account Requisition

New accounts come in from secretary into infrastructure/trunk/acreq/new-account-reqs.txt. There should be entries in infrastructure/trunk/tools/validation/officers/iclas.txt with their information prepended with notinavail. The scripts here process the list of usernames/ full names against iclas.txt, then create ldap, hermes:/home/apmail/.qmail-$availid, update iclas.txt and then send out mail.

Steps!

  • infrastructure/tools/make-accounts.sh Run it about once a day (assuming there are accounts)
  • Pay atttention to output (it can break), sometimes you'll find a username "used" on hermes but not actually anywhere
  • Also watch for mail bounces on new account request forms before running acreq, and make sure to not run acreq for a new poddling before the podling has even requested the resources be setup.
  • Rarely, sometimes the account creation process just breaks. You'll have to undo all the ldap changes and the files created on hermes.
