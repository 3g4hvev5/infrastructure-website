GitHub integration features #

The ASF allows for GitHUb integration with pull requests using a system with the following features:

  • Any Pull Request that gets opened, closed, reopened or commented on can get recorded on the project's mailing list
  • If a project has a JIRA instance, any PRs or comments on PRs that include a JIRA ticket ID will trigger an update on that specific ticket (unless opted out of)
  • Replying to a GitHub comment on the dev@ mailing list will trigger a comment being placed on GitHub
  • GitHub activity can be relayed to IRC channels on the Freenode network.

All these features are opt-in for projects, meaning you will have to file a JIRA ticket to have them added to your project. Please set the component to git when creating a ticket.
